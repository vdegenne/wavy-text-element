# wavy-text-element

<img src="https://raw.githubusercontent.com/vdegenne/wavy-text-element/refs/heads/main/demo.gif" />

## Install

```bash
npm i -D wavy-text-element
```

## Usage

When you need it

```js
// Loads the custom element definition
import 'wavy-text-element'
```

Then in your html, for instance

```html
<wavy-text label="locked" speed="1.5" height="10" delay="60"></wavy-text>
```

(speed in s, height in px, delay between each letter in ms)
All attributes are optional.
